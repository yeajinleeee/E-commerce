## Project Summary
- 사용자에게 기업에 대한 정보와 최근 주가 정보를 제공하여 주식 투자에 도움을 주는 챗봇

**stock_chatbot**
- 팀명 : SOL
- 주제 : 사용자에게 기업에 대한 최신 정보와 최근 주가 정보를 제공하고 기업에 대한 자세한 정보를 QA해주는 챗봇 구현
- 목적/효과 : 주식을 투자하고싶으나 기업에 대한 자세한 정보를 찾기 귀찮은 사용자에게 최신 정보를 제공함
